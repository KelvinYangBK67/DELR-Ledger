# DELR Ledger 版本追蹤

[English](CHANGELOG.md)

## v0.1.2

此修訂版本改善金額輸入與分頁操作體驗。

### 新增

- 在分頁賬本視圖中新增首頁、尾頁，以及直接跳轉頁碼控制。
- 新增繁體中文、英文、德文的跳頁介面文字。

### 調整

- 新增與修改條目的金額輸入框現在使用與剪貼板匯入相同的解析邏輯，支援逗號小數點與千分位分隔符。

### 維護

- 清理工作區中重複的 `(1)` 副本與過期生成/快取副本。

## v0.1.1

`v0.1.0` 之後的第一個修訂版本，主要集中在賬本正確性，以及導入/導出可靠性。

### 修復

- 修正賬本總計，改為使用實際存儲金額的代數和。
- 改善 PDF 導出時中文、中文標點，以及中英混排的字體處理。
- 修正剪貼板匯入時同一行內日期、類型、金額、單位、支付方式被重複識別的問題。
- 改善金額顯示一致性，包括德語介面使用小數逗號。

### 調整

- 文檔導出與表格總計統一遵循實際存儲金額語義。

### 開發者相關

- 改善 Windows 打包腳本，降低 PyInstaller 清理資料夾時遇到權限問題的機率。
- 打包腳本現在會正確回報 PyInstaller 失敗，不再誤顯示成功。
- 打包腳本現在會先清理舊的 `build` 與 `dist` 輸出，再執行 PyInstaller。

## v0.1.0

DELR Ledger 的第一個公開版本。

### 新增

- 用於個人記賬的 Tkinter 桌面 GUI。
- 本地 `.delr` 賬本檔案，內容相容 CSV。
- 多語系介面：繁體中文、英語、德語。
- 賬本新建、開啟、導入、導出流程。
- 剪貼板智能匯入，支援識別日期、類型、金額、單位與支付方式提示詞。
- 支援 `.delr`、`.csv`、`.tsv`、`.xlsx`、`.json`、`.xml`、`.yaml` / `.yml` 導入與導出。
- 支援 Markdown、Word、PDF 文檔導出。
- 表格排序、篩選、按全部/年/月/日分頁，以及多選刪除/修改。
- 按貨幣分別統計總計、收入、支出。
- Windows 打包與發佈腳本。
